# 🗄️ SQL Practice — Window Functions for Data Engineers

A collection of **interactive Jupyter notebooks** for practicing SQL **window functions** using [DuckDB](https://duckdb.org/) and [pandas](https://pandas.pydata.org/). Designed for data engineers who want to master analytical SQL patterns used in real-world pipelines, interviews, and BI work.

---

## 📚 Notebooks

| # | Notebook | Window Functions |
|---|----------|-----------------|
| 00 | `00_setup_and_data.ipynb` | Environment setup, data generation, DuckDB intro |
| 01 | `01_ranking_functions.ipynb` | `ROW_NUMBER`, `RANK`, `DENSE_RANK`, `NTILE`, `PERCENT_RANK`, `CUME_DIST` |
| 02 | `02_offset_functions.ipynb` | `LAG`, `LEAD`, `FIRST_VALUE`, `LAST_VALUE`, `NTH_VALUE` |
| 03 | `03_running_aggregates.ipynb` | `SUM/AVG/COUNT/MIN/MAX OVER`, running totals, rolling windows |
| 04 | `04_advanced_challenges.ipynb` | Multi-level ranking, gaps & islands, market share, anomaly detection |

Each notebook contains:
- 📖 A concise **function reference table**
- ❓ **8 exercises** progressing from basic to challenging
- 💡 **Collapsible solutions** so you can attempt first, then verify

## 📋 Exercise Highlights

### Ranking Functions (NB 01)
- Rank employees by salary using `ROW_NUMBER` vs `RANK` vs `DENSE_RANK` — understand the difference
- Find the **top 3 highest-paid employees per department**
- Assign salary **quartiles** with `NTILE(4)`
- Identify the **top 10% earners** using `PERCENT_RANK`

### Offset Functions (NB 02)
- **Month-over-month** sales comparison and growth rate with `LAG`
- Compare each sale to the rep's **career-first sale** using `FIRST_VALUE`
- Detect the **days between consecutive sales** with `LEAD`
- Find each rep's **3rd sale ever** with `NTH_VALUE`

### Running Aggregates (NB 03)
- **Cumulative (running) total** of daily sales
- **3-month rolling average** revenue
- Each sale as a **% of total** and **% of regional** revenue
- **YTD (year-to-date)** cumulative sales by region
- Salary vs. **department average** (above/below delta)

### Advanced Challenges (NB 04)
- **Top-3 reps per region per quarter** (multi-level partitioned ranking)
- **Consecutive months** above quota (gaps & islands pattern)
- **Product market share** by quarter
- **Anomaly detection** — sales more than 2σ above the rep's mean
- **QoQ revenue growth** rate by region
